# plagiarism_checker_python
A simple plagiarism checker using re, math and tkinter modules
re - quicly returns the result after pattern matching
math - for mathematical functions
tkinter - to create user interface

enter sample text in dtabase1 text
run the code
enter some text in the textbox
click on check plagiarism button. It will compare the entered text with that in database1.txt and display the percentage of plagiarism
click on clear button to clear the text and check plagiarism for more texts
